# Library Management System

## 📌 Project Description

This project is a console-based Library Management System developed using Java. It helps in managing books, students, and book issuance efficiently.

## 🚀 Features

- Add and view books
- Add and view students
- Issue books to students
- Return books
- Persistent storage using file handling

## 🛠️ Technologies Used

- Java (OOP Concepts)
- ArrayList
- File I/O (books.txt, students.txt, issued.txt)

## 📂 Project Structure

src/ → Source code  
docs/ → UML diagrams  
report/ → Final report  
slides/ → Presentation

## ▶️ How to Run

```bash
cd src
javac *.java
java Main
```
